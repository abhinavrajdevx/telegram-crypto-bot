`BOT_TOKEN`



## Run Locally

Clone the project

```bash
  git clone https://github.com/abhinavrajdevx/telegram-crypto-bot
```

Go to the project directory

```bash
  cd telegram-crypto-bot
```

Install dependencies

```bash
  npm install
```
Install dependencies

```bash
  npm install
```

Start the bot

```bash
  npm run start
```


## Telegram Commands

/trending => Get trending coins.

/price bitcoin => Get the price of a coin by its id.

/search btc => search for a crypto and get its id.
